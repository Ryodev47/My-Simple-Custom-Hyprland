# My-Simple-Custom-Hyprland
This is simple hyprland setup for daily driver. A minimal, clean, and functional **Hyprland** setup. This repository serves as a personal backup and a quick-start guide for my desktop environment across different machines.

## 🛠️ Core Components
* **Compositor:** [Hyprland](https://hyprland.org/) (Wayland)
* **Status Bar:** Waybar
* **App Launcher:** Wofi / Rofi
* **Terminal:** Kitty
* **Notification Daemon:** Mako / Dunst
* **Wallpaper Utility:** Hyprpaper / SWWW
